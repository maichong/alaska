# alaska-field-image
Alaska image link field

```javascript
//alaska config

export default {

  'alaska-field-image-link': {
    multi: false,
    //cdn image thumb url suffix, string or empty
    thumbSuffix: '@2o_200w_1l_90Q.jpg',
  }

}
```
