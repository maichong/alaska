# alaska-field-number
Alaska number field

```javascript
//alaska config

export default {

  'alaska-field-number': {
    //numeral format
    format: '$ 0.00',
    min: 0,
    max: [100,'The value of path "{PATH}" ({VALUE}) exceeds the limit ({MAX}).']
  }

}
```
